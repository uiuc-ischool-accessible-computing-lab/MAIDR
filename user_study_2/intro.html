Introduction

This user study intends to investigate how different modalities affect the capacity of blind and low-vision individuals to acquire fundamental statistical concepts via accessible data visualization.

This study is divided into four sections, each of which focuses on a different visualization. In each section, you will be provided with a definition of the visualization and an audio tutorial on how to use Multimodal Access and Interactive Data Representation System (MAIDR) for that particular visualization. After that, you need to complete a task with three to five questions.

Please inform the researcher if you have any questions during the study.
